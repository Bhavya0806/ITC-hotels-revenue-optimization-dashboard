# ITC Hotels Revenue Optimization Dashboard

## 📊 Overview
This project analyzes ITC Hotels’ performance using Power BI to uncover insights into revenue, occupancy, booking behavior, and cancellations.

## 🎯 Objectives
- Analyze financial performance
- Understand occupancy trends
- Evaluate room category performance
- Identify revenue loss due to cancellations

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling

## 📈 Key Insights
- Revenue shows consistent growth
- Weekend occupancy is higher than weekdays
- Premium rooms contribute highest revenue
- Cancellations lead to revenue loss
